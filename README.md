### Employee Management App (Demo)

This project is a work-in-progress employee management system built with ASP.NET Core MVC.
It demonstrates:

- MVC architecture (Controllers, Models, Views)
- Entity Framework Core with code-first migrations
- CRUD operations
- ViewModels for shaping UI data
- Separation of concerns
- Use of helper classes and configuration files

**Project Status**

This project is currently in development, and the code in this repository represents the current working state.
Some parts are still being built, but the core structure and logic are functional and visible.

**Features (so far)**

- Employee list display
- Adding and editing employee data (depending on your progress)
- Database migrations
- Razor Views
- Basic validation
- MVC routing and middleware (Program.cs)

**Future Plans**
- Add authentication
- Add role-based access
- Implement search and filtering
- Add pagination

**Tech Stack**

- ASP.NET Core 8
- C#
- Entity Framework Core
- Razor Views
- SQL (via EF Core Migrations)

**How to Run**

Clone the repository
Restore NuGet packages
Update your database
dotnet ef database update
Run the project via Visual Studio or
dotnet run

### Note

This repository contains only part of the complete application, as the project is still being developed. More code and features will be added over time.
